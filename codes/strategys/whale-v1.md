##  前瞻性偏差测试
该策略经过 前瞻性偏差（Lookahead Bias）分析，结果显示：
+ has_bias: No → 没有前瞻性偏差
+ biased_entry_signals: 0 → 没有前瞻性的入场信号
+ biased_exit_signals: 0 → 没有前瞻性的出场信号
+ biased_indicators: 空 → 没有存在偏差的指标

这意味着策略在回测中`没有使用未来数据`，不会因为使用未来数据而导致回测结果虚假偏高的结果。
